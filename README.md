## 题目要求
题目十三：学生网上选课系统的设计开发  
学生网上选课系统是教学管理中的一种重要应用。系统的基本目标包括：  
1、图形用户界面。  
2、必修课自动选课。  
3、限选课选课功能。  
4、任选课选课功能。  
5、成绩录入功能。  
6、成绩打印功能。
